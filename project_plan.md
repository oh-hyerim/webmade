# 웹메이드 (WebMade) 공식 홈페이지

## 1. 프로젝트 설명
웹메이드는 사업 이해도가 높은 상담 중심의 홈페이지 제작사입니다.
단순 제작이 아닌 맞춤 기획 + 상담 중심으로 포지셔닝하며, 전국 사장님을 대상으로 회사소개형/서비스안내형/상담유도형 홈페이지를 제작합니다.

**타겟 유저**: 현재 사업 운영 중인 전국 사장님, 쇼핑몰 제외 상담형 홈페이지 필요 고객

**핵심 가치**: 신뢰감 형성 → 서비스 이해 → 상담 전환

## 2. 페이지 구조
- `/` - 홈 (메인 배너, 서비스 요약, 차별점, 제작 사례, 후기, 진행 방식, FAQ, CTA)
- `/cases` - 제작 사례
- `/contact` - 문의하기
